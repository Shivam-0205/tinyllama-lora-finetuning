# TinyLlama LoRA Fine-Tuning

This project focuses on fine-tuning TinyLlama-1.1B using LoRA and PEFT on multiple text-only datasets from Hugging Face.

## Features
- LoRA fine-tuning
- PEFT integration
- W&B experiment tracking
- Multi-dataset training
- Evaluation and inference benchmarking

## Model
TinyLlama/TinyLlama-1.1B-Chat-v1.0
